### Object Decomposition Specs
Decomposing specification of real world objects to Java objects.

#### Student
- Long studentId
- String firstName
- String lastName
- Faculty faculty
- List<Course> enrolledCourses
- double tuitionOwed

#### Course
- Long courseId
- String courseName
- String courseCode
- int credit
- BigDecimal costPerCredit
- BigDecimal totalCostOfCourse
